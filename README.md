RaspberryPVR
============

Notes about creating a Freeview PVR from a Raspberry Pi, a cheap DVB-T dongle and RaspBMC software.

--------

*Raspberry Pi model B (original version)
*8GB SD card
*August DVB-T205 USB Freeview Tuner Stick
